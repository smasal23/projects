# Aerial Object Classification & Detection

## 🚀 Project Overview
This project builds an end-to-end system to:
- Classify aerial objects (Bird vs Drone)
- Detect objects using YOLOv8

It combines:
- Deep Learning (CNN + Transfer Learning)
- Object Detection (YOLOv8)
- Deployment (Streamlit)

---

## 🎯 Problem Statement
Distinguish between birds and drones in aerial imagery to support:
- Surveillance systems
- Airspace monitoring
- Security applications

---

## 📊 Dataset Summary
- Classification: Bird vs Drone images
- Detection: YOLO format (bounding boxes)
- Splits: Train / Validation / Test

---

## ⚙️ Workflow
1. Dataset Audit
2. Preprocessing Pipeline
3. Custom CNN Training
4. Transfer Learning
5. Model Evaluation & Selection
6. YOLOv8 Detection
7. Streamlit Deployment

---

## 🧠 Models Used
- Custom CNN
- MobileNet / EfficientNet / ResNet (Transfer Learning)
- YOLOv8 (Detection)
